## Cara seting tampilan OTT seperti Wiseplay:-

- SETTING
- LIST SETTING 
- HOW LIST SHOULD BE DISPLAYED - LARGE CARD - BACK
- LIST POSITION ON SCREEN - CENTER - BACK
- LIST SIZE ON SCREEN - 100%
